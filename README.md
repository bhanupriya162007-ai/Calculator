# Calculator App

A simple and responsive calculator built using **HTML**, **CSS**, and **JavaScript**. This project performs basic arithmetic operations with an easy-to-use interface.

## Features

- Basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Clear (`C`) button to reset the display
- Decimal number support
- Responsive and clean user interface
- Instant calculation using the `=` button

## Project Structure

```
CalculatorApp/
│── index.html      # Structure of the calculator
│── style.css       # Styling and layout
│── script.js       # Calculator functionality
```

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## How to Run

1. Download or clone this repository.
2. Open the `CalculatorApp` folder.
3. Double-click `index.html` or open it in any modern web browser.

No additional setup or installation is required.

## How It Works

- Click the number and operator buttons to create an expression.
- Press **=** to calculate the result.
- Press **C** to clear the display and start a new calculation.
